# Commands that will be added to your CLI by Charmil

```bash
$ yourCLI extensions index
$ yourCLI extensions index list
$ yourCLI extensions index add
$ yourCLI extensions index remove

$ yourCLI extensions info PLUGIN
$ yourCLI extensions info INDEX/PLUGIN

$ yourCLI extensions install NAME
$ yourCLI extensions install < file.txt

$ yourCLI extensions install INDEX/NAME [INDEX/NAME...]
$ yourCLI extensions install --manifest=FILE [--archive=FILE]

$ yourCLI extensions list

$ yourCLI extensions search KEYWORD

$ yourCLI extensions uninstall NAME [NAME...]

$ yourCLI extensions update # update the local copy of the plugin index

$ yourCLI extensions upgrade # upgrade installed plugins to new versions

$ yourCLI extensions version # version of charmil
```
